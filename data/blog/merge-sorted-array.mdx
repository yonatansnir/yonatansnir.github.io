ישנן כמה דרכים לכתוב את האלגוריתם הזה אך במדריך זה אכתוב בצורה הפשוטה ביותר כך שתתאים גם לחבר'ה
שכותבים ב-C. מי שכותב ב-JavaScript בלבד יכול כמובן לשנות את הקוד ולהשתמש בפונקציות המובנות של השפה
(push). נתחיל...

<p>
	נניח וקיבלנו 2 מערכים ממוינים בסדר עולה, אנחנו רוצים לכתוב פונקציה שתקבל את
	המערכים האלה ותמזג אותם למערך אחד ממוין. אנחנו יכולים להבין ישר שחיבור של כל
	האיברים האלה יהיה הגודל של המערך החדש. נתחיל לכתוב את הפונקציה:
</p>

```JavaScript
function mergeSortedArr(arr1, arr2){
  let length = arr1.length + arr2.length;
  const sortedArr = new Array(length);

  return sortedArr;
}
```

<p>
	אנחנו כותבים משתנה <code>length</code> מחזיק לנו את האורך של שני המערכים,
	ואז יוצרים מערך חדש <code>sortedArr</code> באורך של המשתנה שלנו. כתיבה כזאת
	ב-JavaScript שוות ערך לכתיבת <code>sortedArr[length]</code> ב-C.{' '}
</p>

<p>
	עכשיו אחרי שיש לנו את המערך בגודל הדרוש, אנחנו יכולים להתחיל לרוץ על שני
	המערכים ולחבר בניהם. אנחנו כבר יודעים שהלולאה שלנו צריכה לרוץ{' '}
	<code>length</code> פעמים, פעם אחת על כל מיקום במערך החדש.{' '}
</p>

<p>
	מה שנשאר לנו זה ליצור שני משתנים חדשים שיחזיקו לנו את האינדקס של כל מערך.
	בכל איטרציה לבדוק איזה מבין האיברים של שני המערכים שאנחנו עומדים עליהם קטן
	יותר, ואותו להכניס למערך החדש. לבסוף, לקדם את האינדקס של אותו מערך באחד.
</p>

```JavaScript
let p1 = 0, p2 = 0; // the indexes of the arrays

  for (let i = 0; i &lt; length; i++){
    if (arr1[p1] &lt; arr2[p2]){ // Which one is bigger?
      sortedArr[i] = arr1[p1];
      p1++;
    } else {
      sortedArr[i] = arr2[p2];

    }

```

<p>והפונקציה במלואה:</p>

```JavaScript
function mergeSortedArr(arr1, arr2){
  let length = arr1.length + arr2.length;
  const sortedArr = new Array(length);
  let p1 = 0, p2 = 0; // the indexes of the arrays

  for (let i = 0; i &lt; length; i++){
    if (arr1[p1] &lt; arr2[p2]){ // Which one is bigger?
      sortedArr[i] = arr1[p1];
      p1++;
    } else {
      sortedArr[i] = arr2[p2];
      p2++
    }
  }

  return sortedArr;
}
```

	לשאול בתגובות.
</p>
